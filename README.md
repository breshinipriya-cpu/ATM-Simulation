# ATM Simulation System

## 📌 Description
A Java-based ATM Simulation project that allows users to perform banking operations such as balance inquiry, deposit, withdrawal, and transaction history.

## 💻 Technologies Used
- Java
- OOP Concepts

## ⚙️ Features
- User authentication (PIN-based)
- Deposit and withdrawal
- Balance checking
- Transaction handling

## 🚀 How to Run
1. Compile:
   javac *.java
2. Run:
   java ATM

## 📂 Project Structure
- ATM.java
- Account.java
- User.java
